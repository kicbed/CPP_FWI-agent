# Cycle Skipping (周波跳跃) 问题

## 一、定义

**Cycle skipping**（周波跳跃）是 FWI 中最经典的非线性问题之一，指模拟波形与观测波形之间的走时差超过半个周期（$\Delta t > T/2$），导致互相关主瓣跳到相邻周期。

## 二、物理机制

### 2.1 数学描述

设观测波形为 $d^{obs}(t)$，模拟波形为 $d^{syn}(t)$，两者的时间延迟为 $\Delta t$。

当 $|\Delta t| < T/2$ 时：
- 互相关主峰在零延迟处
- 梯度指向正确的更新方向

当 $|\Delta t| > T/2$ 时：
- 互相关主峰跳到 $\pm T$ 处
- 梯度指向错误的更新方向
- 反演陷入局部极小

### 2.2 经验判据

若最大走时偏差满足：
$$
\max|\Delta t| > \frac{1}{2 f_{dom}}
$$

其中 $f_{dom}$ 为数据主频，则大概率发生 cycle skipping。

**示例**:
- 主频 10 Hz → $\Delta t_{max} = 50$ ms
- 地下 3 km 深度，速度 3000 m/s → 走时 1000 ms
- 50 ms 走时误差 → 约 150 m 位置偏差

## 三、典型表现

| 现象 | 说明 |
|------|------|
| 残差波形呈近似整周期平移 | 观测与模拟波形形状高度相似，但整体左/右滑动 N 个波峰 |
| 梯度方向与地质趋势相反 | 如已知某区域应为高速体，梯度却强烈指示降低速度 |
| 多尺度反演中高频无法收敛 | 低频（< 5 Hz）能收敛，引入 8–15 Hz 后目标函数震荡或突变上升 |
| 反演结果出现"阶梯状""条带状"伪影 | 尤其在层状介质中，界面位置系统性上移/下半波长 |

## 四、产生原因

| 原因 | 说明 | 典型场景 |
|------|------|----------|
| **低频缺失** | 数据缺乏 < 5–7 Hz 成分 → 波形包络模糊，无法约束长波长结构 | 地震勘探中近地表吸收强、仪器低频响应差 |
| **初始模型太粗糙** | 初始速度偏差 > 5–10% | 盆地/盐下/火成岩区建模困难 |
| **强速度对比/复杂构造** | 断层、盐丘、高速基底等导致多次波、绕射波混叠，波形非单调 | 墨西哥湾盐丘、塔里木盆地深层 |

## 五、应对策略

### 5.1 多尺度反演 (Multiscale FWI)

**原理**: 从极低频（~2–3 Hz）开始反演，逐步加入高频。

**流程**:
1. 先用 2–5 Hz 数据更新背景速度
2. 再用 5–10 Hz 数据细化构造
3. 最后用 10–15 Hz 数据恢复细节

**优点**: 工业标准流程，鲁棒性强
**局限**: 依赖低频能量，野外常缺乏 < 3 Hz 有效信号

### 5.2 包络反演 (Envelope Inversion)

**原理**: 最小化波形包络（Hilbert envelope）残差：

$$
J_{env} = \frac{1}{2} \sum_{s,r} \left\| |d^{obs}_{s,r}(t)| - |d^{syn}_{s,r}(t)| \right\|^2
$$

其中 $|d(t)| = \sqrt{d^2(t) + \hat{d}^2(t)}$，$\hat{d}$ 为 Hilbert 变换。

**优点**: 对相位不敏感，容忍走时差达数个周期
**局限**: 分辨率较低，仅恢复宏观结构

### 5.3 自适应波形反演 (AWI)

**原理**: 动态加权残差，抑制振幅匹配、强化走时对齐：

$$
J_{AWI} = \frac{1}{2} \sum_{s,r} \left\| w_{s,r}(t) \cdot (d^{obs}_{s,r}(t) - d^{syn}_{s,r}(t)) \right\|^2
$$

权重 $w_{s,r}(t)$ 基于互相关延迟或振幅信噪比动态调整。

**优点**: 更强抗 cycle skipping 能力
**局限**: 实现复杂，计算开销增加约 30%

### 5.4 正则化 + 先验约束

**原理**: 引入平滑约束（Tikhonov）、TV 正则化，或利用井数据/地质解释构建参考模型：

$$
J_{reg} = J + \frac{\beta}{2} \|\mathbf{m} - \mathbf{m}_{ref}\|^2
$$

**优点**: 抑制高频噪声与非物理解
**局限**: 过度平滑损失细节，先验不准则引入偏差
